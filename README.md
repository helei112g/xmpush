# 概述

目前国内消息推送，从稳定性、资费、到达率等标准来衡量，个人推荐 **小米推送** 。

由于小米官方没有提供composer的安装渠道。对于现代化的开发，非常不方便。

个人简单将小米推送代码加入一些composer信息后，可使用composer进行安装。
